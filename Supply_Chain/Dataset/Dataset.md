# Dataset Source

Dataset: Supply Chain Shipment Pricing Data  
Source: Kaggle  
Link: https://www.kaggle.com/datasets/divyeshardeshana/supply-chain-shipment-pricing-data  
Records: 10,000+ shipments  

Attributes include:
- Shipment dates  
- Vendors  
- Countries  
- Shipment mode  
- Quantity  
- Shipment value  
- Freight cost
